# Skills — Index

Reusable workflow docs. Each skill file documents a repeatable workflow (e.g., creator outreach, voice clone setup, prompt patterns).

## What belongs here
- Step-by-step workflows used more than once.
- Prompt patterns, templates, recipes.
- Setup/operational guides for tools and integrations.

## What does not belong here
- Project state → `.claude/context.md`
- Brand/tone → `.claude/brand.md`
- One-off decisions → `.claude/decisions.md` (create when earned)

## Rule
- Add a skill file only when the workflow has been written and used. Don't pre-create.
- When a new skill is added, list it below and log the addition in `changelog.md`.
